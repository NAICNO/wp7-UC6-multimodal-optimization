Multi-Modal Optimization with SHGA
==================================

.. important::
   This tutorial teaches you how to run multi-modal optimization experiments
   on NAIC Orchestrator VMs using the Scalable Hybrid Genetic Algorithm (SHGA).

The SHGA algorithm finds **multiple** local and global optima in continuous
optimization problems, combining genetic algorithms (global search) with
CMA-ES (local refinement).
